PyInstaller GUI

A simple GUI application for PyInstaller. This GUI uses the PyInstaller command line application to turn .py files to .exe programs.

BSCode

PyShell Discord Bot

Discord bot that can run Python code in a Discord channel.

PyInstaller GUI
PyInstaller GUI is a simple easy-to-use graphic user interface program that uses the PyInstaller command line application. Turn your Python programs into executable programs, without the need to use the terminal.

BSCode
BSCode, named derived from but not related to Microsoft's Visual Studio Code (VSCode), is a lightweight text editor for code. Able to edit any language, it can also natively run Python, Node.js, and Swift; and display HTML.

PyShell Discord Bot
The PyShell Discord Bot opens a Python shell right in your Discord server. Name a server "#pyshell", and start typing Python code.
